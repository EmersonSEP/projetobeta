PWD mostra qual pasta estamos
LS mostra os arquivos na pasta
touch comando para criar arquivo
git init - inicia repositorio local
git config - colocar nome e email
git status - mostra o status do nosso Git
git add +nome arquivo - adiciona arquivo no proximo commit
git commit -m 'mensagen'
Se for a primeira vez vai ter que colocar email e nome. Usando copiar e colar, colocando seus dados.
git . - Voce atualiza todos os arquivos que foram alterados
